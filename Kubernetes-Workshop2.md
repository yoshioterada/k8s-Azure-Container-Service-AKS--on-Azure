```
$ az aks scale --name esakscluster --resource-group MCACS-AKS --node-count 7
```

```
$ az aks upgrade --name esakscluster --resource-group MCACS-AKS  --kubernetes-version 1.8.7 --yes
 - Running ..

..... take a long time (over 40 min) 
```
